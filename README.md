# Power-BI-Sales-Dashboard
Complex Power BI Sales Dashboard , focusing on sales metrics using heavy DAX like : YOY% growth etc  

Interactive Data Dashboard

An interactive dashboard designed to visualize business performance metrics, trends, and key insights using real-world data. This project demonstrates strong data analysis, visualization, and storytelling skills.

 Project Overview

This dashboard provides insights into:

Sales and revenue trends

Performance across time periods

Regional and category-level distribution

Key performance indicators (KPIs)

The objective is to enable stakeholders to quickly understand performance patterns and support data-driven decision-making.
